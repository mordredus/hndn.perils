# HN-DN Official Concept Website

**Leafless Tea — May 14 Release**  

This repository hosts the **HN-DN fan/official concept website**, showcasing:

- Albums and release information  
- Wuxing-inspired member bios  
- Exclusive pre-save teaser section  

The website is **black-and-white minimalist**, designed for a sleek editorial aesthetic, using the **Hotel Oriental** font.  

---

## Features

### Navigation
- **Albums**: Clickable album tiles display release date, genre, and concept.  
- **Members**: Wuxing-inspired bios with symbolic attributes (Wood, Fire, Earth, Metal, Water).  
- **Pre-Save**: Interactive “unlock” button reveals exclusive teaser content.  

### Design
- Sleek black-and-white theme  
- Hotel Oriental font for premium editorial look  
- Hover effects and subtle animations  

### Technology
- HTML5  
- CSS3  
- JavaScript (vanilla)  
- Fully static — works on **GitHub Pages**

## Usage

- Open the site in any modern browser.  
- Navigate using the top buttons to explore **Albums**, **Members**, and **Pre-Save**.  
- Click album tiles to see dynamic info.  
- Click member names to view Wuxing-inspired bios.  
- Unlock the pre-save teaser with the button.
